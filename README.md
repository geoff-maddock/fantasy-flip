# Fantasy Flip
A fantasy themed flip and write tabletop game.
This repository contains the game design documents, rules, and assets for the game.

# Documentation
- assets - Concrete assets use for the game - board designs etc.
- context - Initial context used to help design the game
    /prompts - AI prompts used for generating game content and ideas.
    /inspirations - Images and other media that inspired the design of the game.
- versions - Concretized versions of the game rulesets.  Seperate from version control, these are intended to be shared with playtesters.
